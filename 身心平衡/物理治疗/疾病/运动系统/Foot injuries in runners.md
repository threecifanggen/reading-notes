# 正文分析

## 趾骨受伤

标签: #sesamoid; #趾骨; 

* 5% ~ 33%的人群有趾骨损伤
* 多数趾骨损伤与过度使用有关
  * 压力 40%
  * 趾骨炎 30%
  * 急性骨折 10%
  * 骨软骨炎( #osteochondritis ) 10%
  * 骨关节炎( #osteoarthritis ) 5%
  * 粘液囊炎 ( #bursitis ) 5%
* 在足部受压时，就会感到疼痛，这些病人会选择用侧边腿站立来防止疼痛

## 裸足跑

裸足跑更容易发生前足和中部足的损伤；穿鞋跑是后部
