 (Kindle 位置 247-248)

| 梦的现象     | 精神分析             | 激活-整合                 |
| ------------ | -------------------- | ------------------------- |
| 诱因（动机） | 被压抑的潜意识愿望   | 睡眠期间的大脑激活        |
| 视觉表象     | 退至感觉层面         | 高级视觉中枢激活          |
| 错觉         | 初级加工思维         | DLPFC失活导致工作记忆丧失 |
| 怪诞         | 愿望的伪装           | 超联想的整合              |
| 情感         | 对自我的次级防御反应 | 边缘系统的原发激活        |
| 遗忘         | 压抑                 | 器质性（生理性）失忆      |
| 意义         | 相当含糊不清         | 清晰，明显                |
| 解析         | 需要                 | 不需要                    |
