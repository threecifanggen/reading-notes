## 恋尸癖的十种主要分类

标签: #恋尸癖

作者基于此指定恋兽癖分类

![image-20210227113525354](https://typora-picgo-bed.oss-cn-beijing.aliyuncs.com/image-20210227113525354.png)

| 类别 | 名称             | 主要特征                                                     |
| :--- | :--------------- | ------------------------------------------------------------ |
| I    | 角色扮演         | 不想和尸体性交。喜欢和一个装作尸体的人性交                   |
| II   | 浪漫恋尸         | 只和一个喜欢的人的尸体在一起，排斥其他尸体                   |
| III  | 恋尸幻想         | 幻想与尸体性交。可能参加葬礼并在尸体上自慰                   |
| IV   | 接触恋尸         | 喜欢接触尸体。喜欢打击尸体的部位比如胸部                     |
| V    | 恋物性恋尸       | 切割尸体的部分保存，比如胸部。但和II不同是可以和陌生人发生   |
| VI   | 肢解恋尸         | 不止喜欢接触，而且喜欢肢解                                   |
| VII  | 机会主义式的恋尸 | 由于某次偶然机会发生一些性行为。有性行为的恋尸从这个分类开始。 |
| VIII | 频繁恋尸         | 「经典」的恋尸癖。不太喜欢和活人性交，更喜欢和尸体。不过不排斥和活人性交 |
| IX   | 杀人恋尸         | 通过杀人得到尸体满足恋尸。可以和活人性交，但是没法满足尸体带来的快感。 |
| X    | 排斥性恋尸       | 只喜欢和尸体性交。完全无法接受和活人性交。                   |
